# CubeWorld
Graphical implementation using Voxel Traversal Algorithm

# Key Set

-Shortcut-
ESC: pause

-Movement-
Moving: W,A,S,D
Jump: V
Dash: Q

-Map Interaction-

Add Block: Space Bar
Delete Block: X
Shoot Paint: E,R,T

-Graphic Quality Setting-
N: Better quality, Lower performance
M: Better performance, Lower quality
